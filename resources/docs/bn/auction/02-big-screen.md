# Big screen

Big screen হলো audience display। Projector বা TV-তে current player, bid, timer, leading team এবং result animation দেখা যায়।

## Setup

1. Dashboard → Auction → Big Screen খুলুন
2. Display laptop projector/TV-তে connect করুন
3. Browser fullscreen করুন
4. Test bid দিয়ে realtime update দেখুন

## ভালো display এর জন্য

- Laptop charger connected রাখুন
- Sleep mode off করুন
- 100% zoom বা fullscreen ব্যবহার করুন
- Player photo clear রাখুন

## Update না হলে

Internet, websocket console error এবং Reverb service status check করুন।
