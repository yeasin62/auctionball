# Team bidding

Team owner বা captain নিজের private team device link থেকে bid করে। এই link admin tool দেখায় না, শুধু current auction information এবং bid button দেখায়।

## Team link share

- Teams page থেকে প্রতিটি team-এর device link copy করুন
- প্রতিটি team-কে শুধু নিজের link দিন
- Publicly সব link share করবেন না

## Captain যা দেখবে

- Current player
- Current bid
- Remaining budget
- Timer
- Valid bid button

## Bid fail হলে

- Team budget যথেষ্ট আছে কিনা দেখুন
- Auction timer running কিনা check করুন
- Correct active season কিনা confirm করুন
- Phone internet issue হলে refresh করুন
