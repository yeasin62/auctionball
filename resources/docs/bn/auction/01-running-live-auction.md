# লাইভ অকশন চালানো

Live auction page হলো auctioneer-এর main control room। এখান থেকে player select, timer start, bid monitor এবং Sold/Unsold করা হয়।

## Auction শুরু করার আগে

- Active season ঠিক আছে কিনা check করুন
- Player queue approve করা আছে কিনা দেখুন
- Team budget ঠিক আছে কিনা confirm করুন
- Big screen ও team device realtime update test করুন

## Player select করা

Left queue থেকে next player select করুন। Player select না করলে timer start হবে না।

## Timer control

- **Start**: bidding শুরু
- **Pause/Resume**: temporary stop/start
- **Reset**: current lot clear
- **Sold**: player winning team-এ assign
- **Unsold**: player unsold হিসেবে mark

## Bid handling

Team device বা control panel থেকে valid increment অনুযায়ী bid আসে। SOLD দেওয়ার আগে leading team ও amount confirm করুন।

## Event-day tips

- একই সময়ে এক person control চালাবে
- SOLD দেওয়ার আগে announce করুন
- Internet issue হলে pause ব্যবহার করুন
- Result export করার আগে roster review করুন
