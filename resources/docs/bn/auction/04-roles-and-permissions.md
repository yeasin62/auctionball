# Role ও permission

সব user-কে admin access দেওয়ার দরকার নেই। কাজ অনুযায়ী role দিন।

## Organization admin

Settings, billing, users, seasons, players, teams এবং auction setup manage করতে পারে।

## Auctioneer

Live auction control চালাতে পারে। Event-day operation এর জন্য এই role useful।

## Team owner

নিজের team link বা account থেকে bid করতে পারে।

## Best practice

- Event এর আগে সব operator login test করুক
- Admin count সীমিত রাখুন
- Team owner-কে admin role দেবেন না
- Suspicious user remove করুন
