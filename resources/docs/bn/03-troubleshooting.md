# সাধারণ প্রশ্ন

সবচেয়ে বেশি যে প্রশ্নগুলো আসে সেগুলো এখানে দেওয়া হলো। আপনার প্রশ্ন না থাকলে [support@auctionball.com](mailto:support@auctionball.com) এ email করুন।

## Setup

> **Organization name taken দেখাচ্ছে। কী করব?**  
Subdomain একটু বদলে দিন, যেমন `bpl-cup` এর বদলে `bpl-cup-2026`।

> **একই সময়ে multiple tournament চালাতে পারব?**  
Pro/Enterprise plan-এ multiple active season parallel চালানো যাবে। Free/Starter-এ live auction সাধারণত এক active season থেকে চলে।

> **Tournament delete করে নতুন করে শুরু করা যাবে?**  
হ্যাঁ। Settings থেকে organization delete করা যায়। আগে দরকারি data export করে নিন।

## Money

> **BDT bid step set করেও dollar দেখাচ্ছে কেন?**  
আপনার display currency USD হলে USD increment আলাদা set করতে হবে অথবা BDT mode-এ ফিরতে হবে।

> **Team budget negative হবে?**  
সাধারণভাবে valid bid budget-এর বেশি হলে reject হয়। ভুল data দেখলে team budget edit করে review করুন।

## Auction night

> **Big screen realtime update হচ্ছে না।**  
Connection indicator, internet এবং Reverb/websocket status check করুন। দরকার হলে refresh করুন।

> **Start button কাজ করছে না।**  
প্রথমে left queue থেকে player select করতে হবে।

> **Sold ভুল হয়েছে, undo করা যাবে?**  
Player select করে Reset দিলে bid clear হয়ে player queue-তে ফিরতে পারে।

## Player registration

> **Form-এ unnecessary field আছে।**  
Seasons → Form builder থেকে custom field remove/reorder করুন।

> **Submitted player list-এ আসছে না।**  
Players page-এর Pending filter check করুন। Public registration approve করতে হয়।

## Billing

> **bKash payment approve হয়নি।**  
Transaction ID সহ support-এ email করুন।

## Login

> **Password ভুলে গেছি।**  
Login page থেকে Forgot password ব্যবহার করুন।

## Language

> **Bengali/English switch করব কীভাবে?**  
Header বা dashboard-এর language toggle ব্যবহার করুন।
