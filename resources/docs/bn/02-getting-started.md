# শুরু করা

এই guide হলো “আমি AuctionBall সম্পর্কে জানলাম” থেকে “আমার account ready, এখন team add করতে পারি” পর্যন্ত দ্রুত পথ।

## Step 1 — Sign up

1. Home page খুলে **Start free** বা **Get started** চাপুন।
2. Fill করুন:
   - **Tournament / organization name** — যেমন “Dhaka Premier League 2026”
   - **Subdomain** — আপনার tournament-এর short web address
   - **Your name and email** — admin account হবে
   - **Password** — কমপক্ষে ৮ character strong password
   - **Plan** — setup করার সময় Free দিয়ে শুরু করতে পারেন
3. **Create organization** চাপুন।

এখন আপনি dashboard-এ login অবস্থায় থাকবেন।

## Step 2 — Currency নির্বাচন করুন

Default ভাবে price, bid ও budget BDT (৳) দেখায়। USD চাইলে:

1. Left sidebar থেকে **Settings** খুলুন
2. **Display currency** থেকে USD select করুন
3. Conversion rate দিন
4. **Save** চাপুন

পরে যেকোনো সময় BDT-তে ফিরতে পারবেন।

## Step 3 — Sport নির্বাচন করুন

প্রথম season তৈরি করার সময় cricket বা football select করুন। Cricket হলে batting/bowling field দেখা যাবে, football হলে আলাদা field দেখাবে।

## Sign up করার পর যা তৈরি থাকে

- Current year-এর জন্য empty season
- Free plan limit
- Browser language অনুযায়ী preferred language

## সাধারণ প্রশ্ন

> **Technical knowledge দরকার?**  
না। সব point-and-click।

> **Multiple admin রাখা যাবে?**  
হ্যাঁ, Users থেকে co-admin invite করা যাবে।

> **Free plan ছোট হয়ে গেলে?**  
Billing থেকে upgrade করুন। Data হারাবে না।
