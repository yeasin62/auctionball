# bKash payment

Manual bKash payment হলে payment করার পর transaction ID submit করতে হয়।

## কী করবেন

1. Billing page থেকে plan select করুন
2. Instruction অনুযায়ী bKash payment করুন
3. Transaction ID copy করুন
4. Billing form-এ ID submit করুন

## Approval

Admin payment verify করলে plan activate হবে। ২৪ ঘণ্টার বেশি delay হলে support-এ transaction ID সহ email করুন।

## Tips

- ভুল account/number এ payment করবেন না
- Screenshot সংরক্ষণ করুন
- Organization slug support message-এ লিখুন
