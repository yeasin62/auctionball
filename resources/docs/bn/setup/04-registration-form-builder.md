# Registration form builder

Form builder দিয়ে public player/team registration form customize করা যায়।

## Field type

- Text
- Number
- Select
- Checkbox
- Image upload
- Payment field
- Section heading

## Best practice

- শুধু দরকারি field রাখুন
- Payment instruction পরিষ্কার লিখুন
- Required field বেশি করবেন না
- Event এর আগে form submit করে test করুন

## Approval flow

Submitted player বা team admin review করে approve করলে main list-এ আসে।
