# Player যোগ করা

Player manual add, public registration approve বা CSV import দিয়ে যোগ করা যায়।

## Manual add

Players page থেকে Add player চাপুন এবং name, category, base price, position ও photo দিন।

## CSV import

বড় list হলে template download করে CSV fill করুন। Required column যেমন `name` ঠিক থাকতে হবে।

## Public registration

Player নিজে form submit করলে admin approve করার পর queue-তে যাবে।

## Photo

Clear square photo ব্যবহার করুন। Big screen ও roster page বেশি professional দেখাবে।
