# Team যোগ করা

Team হলো auction-এর buyer। প্রতিটি team budget নিয়ে player কিনে।

## Add team

1. Dashboard → Teams খুলুন
2. Add team চাপুন
3. Team name ও short code দিন
4. Budget confirm করুন
5. Logo থাকলে upload করুন

## Short code

Short code big screen-এ সুন্দর দেখায়। যেমন Dhaka Dynamites = DHK।

## Team owner

Team owner assign করলে সেই user নিজের team device থেকে bid করতে পারবে।

## Budget check

Auction শুরু করার আগে সব team-এর initial ও remaining budget review করুন।
